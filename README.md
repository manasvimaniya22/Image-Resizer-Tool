# Image Resizer Tool

## Objective
Resize and convert multiple images in batch using Python.

## Tools Used
- Python
- Pillow
- OS Module

## Features
- Reads all images from a folder
- Resizes images to 800x600
- Converts JPG/JPEG to PNG
- Saves output in a separate folder
- Handles errors using try-except

## How to Run
1. Install Pillow: pip install Pillow
2. Put images inside "images" folder
3. Run: python image_resizer.py
4. Check resized images inside "output" folder
